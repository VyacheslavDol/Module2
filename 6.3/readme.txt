Задание 6.3 2 Модуль
Доработать решение задачи 2.3 (калькулятор) так, чтобы
функции загружались из динамических библиотек. В одной библиотеке
находится одна функция. При запуске программы считывается каталог с
библиотеками и загружаются найденные функции.

в каталоге /library/source находятся файлы реализации для функций
в /library находятся заголочные файлы функций, Makefile для создания динамических библиотек

в /project находится основной код программы (task2_3.c), Makefile для компиляции общей программы.
